WDI Project 1 - Create a game in the browser

Battleship

Battleship is a popular turn based game where the aim of the game is to sink your opponent's ships before they sink yours. In this version of Battleship, you'll play against the computer and your ships will be randomly placed for you at the start of the game.

The game's code can be split into 4 sections:

  1. Creating the grids
  2. Placing the ships on the grids
  3. The battle system
  4. The AI

Tech used

  • HTML/CSS
  • JavaScript/jQuery

WDI Project 2 - A Full Stack RESTful application with authentication

WDI Project 3 - A MEAN Stack application with an API created as a team

Runch

RUNCH is an app targeted at getting people, who have little spare time, active. It enables them to use their lunch-time wisely - they can go for a quick run and meet other like-minded professionals. It is an opportunity to not only get fit, but also to network, and feel refreshed for the second half of the day.

WDI Project 4 - A Fullstack Rails application with an API and an SQL database

Henstagenda

Henstagenda is a website that helps you organise stag and hen dos. You can create a group for your stag/hen do, create the agenda of events for your party and invite your friends to it. The group enables everyone to keep up to date with the agenda and find out which friends are able to make it to events by providing their attendance status.

The project helped me improve my understanding of Ruby on Rails through creating custom routes and custom methods that needed to be created and used in conjunction with the database in order to pass meaningful data onto the client side.

Tech used

  • Ruby on Rails
  • PostgreSQL
  • AngularJS
  • BCrypt
  • HTML/CSS/SCSS
  • JavaScript
