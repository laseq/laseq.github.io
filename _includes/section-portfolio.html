WDI Project 1 - Create a game in the browser

Battleship

Battleship is a popular turn based game where the aim of the game is to sink your opponent's ships before they sink yours. In this version of Battleship, you'll play against the computer and your ships will be randomly placed for you at the start of the game.

The game's code can be split into 4 sections:

  1. Creating the grids
  2. Placing the ships on the grids
  3. The battle system
  4. The AI

Tech used

  • HTML/CSS
  • JavaScript/jQuery

Github: https://github.com/laseq/wdi-project-1

Project: https://protected-waters-78340.herokuapp.com

WDI Project 2 - A Full Stack RESTful application

Kanji Kaigou

My project of choice was a web app to enable learners of Japanese to improve their Kanji (Japanese symbol) competence. As a learner of the Japanese language I felt that it would be an exciting project to embark on. Such an app would fulfil the project requirements with RESTful routes and include some logic for lesson reviews.

Tech used

  • Express.js
  • Node.js
  • MongoDB & Mongoose
  • BCrypt
  • HTML/CSS/SCSS
  • JavaScript/jQuery

Github: https://github.com/laseq/wdi-project-2

Project: https://whispering-harbor-84338.herokuapp.com
